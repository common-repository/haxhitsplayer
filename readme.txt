== Description ==

Haxhitsplayer is a plugin to get direct link google drive, mp4upload, xvideos to haxhits player

== Live Demo ==

Live demo: https://haxhits.com/

== Live Support ==

https://www.facebook.com/haxhits/

== Installation ==

1. Upload `wp-player-haxhits.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Settings -> Haxhits Player

== Frequently asked questions ==

Q: Plugin is not working
A: Please disable other players plugin
